

A collection of responsive React components to use with bootstrap .
===================================================================

Requirements:
====================================================================
Requires next.js , bootstrap 5 , nodemailer for the contact form . 
Run npm install 

Components : 
====================================================================
- Banners (fixed and sliding in and out)
- Contact forms
- Stacked cards (arrows and animated)
- Image carousel (with text, or centered) 
- Image with text 
- Hamburger menu 
- Navbars (modern , classic, with cart) 
- Animated Background Canvas

Instructions:
=====================================================================
1. Create a folder(route) in app with a page.tsx 

2. Add your desired navbar in layout.tsx 

Inside page.tsx :
--------------------

1. Import required types 
2. Import wanted components and place them 


NOTES 
===========
Styles are imported into layout.tsx 
Navbar and the bootstrap wrapper too
